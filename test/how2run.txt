C++: 
g++ -o (output file name) (source file name)
./(output file name)

Rust:
cargo run
also "cargo run --bin (bin name: found in cargo.toml)

Typescript:
tsx (source file name)

Go:
go run (source file name)

C#:
dotnet run

Java
javac (source file name, or the entire directory with *).java
java (source file name, no extension)

Julia:
julia (source file name)

Fortran:
gfortran (file name) -o (name of executable)
-o option specifies name of output executable file, else default is a.out
./(name of executable)
